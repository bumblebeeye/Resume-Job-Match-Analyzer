# Frontend (Next.js App Router)

Implemented in Phase 2:
- `/` home page
- `/analyze` analyzer workflow page

Planned for Phase 3:
- `/history`
- `/history/[id]`

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Create `.env.local` and set:
   ```bash
   NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
   ```
3. Start the app:
   ```bash
   npm run dev
   ```
